Simple and fast class, to use the range in the range-based loop.


```cpp
for (auto i: range(N))
{
    cout << i;
}
```

```cpp
for (auto i: range_z(N))
{
    cout << i;
}
```

```cpp
for (auto i: range(N, M))
{
    cout << i;
}
```

```cpp
for (auto i: range(N, M, step))
{
    cout << i;
}
```

```cpp
for (auto i: range(M, N, -step))
{
    cout << i;
}
```
